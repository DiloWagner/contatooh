# Contatooh
Projeto de estudo com a MEAN Stack

### Tecnologias utilizadas
M - MongoDB
E - Express
A - AngularJS
N - Node

Este projeto de estudo, monta um cadastro básico de contatos utilizando estas tecnologias.
